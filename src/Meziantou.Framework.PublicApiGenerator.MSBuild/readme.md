# Meziantou.Framework.PublicApiGenerator.MSBuild

`Meziantou.Framework.PublicApiGenerator.MSBuild` generates a public API file during `dotnet build`.

## Usage

Add the package reference and configure the required output path property:

```xml
<ItemGroup>
  <PackageReference Include="Meziantou.Framework.PublicApiGenerator.MSBuild" Version="x.y.z" PrivateAssets="all" />
</ItemGroup>

<PropertyGroup>
  <PublicApiGeneratorOutputPath>PublicApi/PublicApi.g.cs</PublicApiGeneratorOutputPath>
</PropertyGroup>
```

## Configuration

- `PublicApiGeneratorGenerateOnBuild` (default: `true`)
- `PublicApiGeneratorOutputPath` (**required**)
- `PublicApiGeneratorFileLayout` (`SingleFile`, `OneFilePerNamespace`, `OneFilePerType`; default: `SingleFile`)
- `PublicApiGeneratorIncludeAutoGeneratedComment` (default: `true`)

For multitargeted projects, generation runs once after all target frameworks are built and merges all output assemblies.
