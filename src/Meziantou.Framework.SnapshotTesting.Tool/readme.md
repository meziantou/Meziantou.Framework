# Meziantou.Framework.SnapshotTesting.Tool

`Meziantou.Framework.SnapshotTesting.Tool` approves snapshots created by `Meziantou.Framework.SnapshotTesting` by renaming `*.actual.*` files to `*.verified.*`.

## Install

```bash
dotnet tool install --global Meziantou.Framework.SnapshotTesting.Tool
```

## Example

```bash
Meziantou.Framework.SnapshotTesting.Tool approve
```

```bash
Meziantou.Framework.SnapshotTesting.Tool approve --interactive
```

### approve

```
Description:
  Approve snapshots by renaming *.actual.* files to *.verified.*

Usage:
  Meziantou.Framework.SnapshotTesting.Tool approve [options]

Options:
  --interactive      Prompt for each snapshot to approve or reject
  --folder <folder>  Folder to process. Uses current directory when not set.
  --recurse          Allow recursive search for __snapshots__ folders
  -?, -h, --help     Show help and usage information
```
